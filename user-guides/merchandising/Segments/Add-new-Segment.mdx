## 

To create a segment:

1. Go to _Audience \> Segments \> Create Segment_

2. In the Create Segment page, at the **Select contact type** option, select the List(s)/Segment(s) or Contact set out of which you want to create the segment.

3. Provide the name of your segment at the **Untitled segment** box on top

4. Select the conditions of \<\<glossary:segmentation\>\> as required. Select **AND** or **OR** based on your requirement.

5. To add a block, click **ADD BLOCK**.

6. Click on the dropdown to select an **attribute** or **event**.
