# Netcore Unbxd Analytics Overview

Netcore Unbxd Analytics integrates with your website to track key events like:

- Search queries
- Product clicks
- Impressions
- Cart additions
- Orders

<Info>
  These events enable AI/ML models to generate insights, build visitor profiles,
  and deliver personalized search results, recommendations, and reports.
</Info>

<AccordionGroup>
  <Accordion title="Why is it essential to set up Netcore Unbxd Analytics?">
    <p>Integrating Analytics helps you achieve conversion goals by:</p>
    <ul>
      <li>Delivering personalized search results</li>
      <li>Enhancing product recommendation algorithms</li>
      <li>Generating detailed reports</li>
    </ul>
    <Info>
      Incomplete analytics integration leads to diminished search experience,
      reduced relevance, and ineffective AI capabilities.
    </Info>
  </Accordion>
</AccordionGroup>

# Events We Track

<AccordionGroup>
  <Accordion title="Visitor Event" description="Tracks visitor profiles">
    <p>Triggered when a shopper visits your site, tracking:</p>
    <ul>
      <li>User ID: <code>unbxd.userId</code></li>
      <li>Visit ID: <code>unbxd.visitId</code></li>
      <li>Visit Type: <code>unbxd.visit</code></li>
    </ul>
  </Accordion>

{" "}

<Accordion title="Search Hit" description="Triggered by search activity">
  <p>
    Records every search query, including no-result queries, providing insights
    into search effectiveness.
  </p>
</Accordion>

{" "}

<Accordion title="Product Click" description="Tracks product interactions">
  <p>Triggered when a shopper clicks a product or its image on:</p>
  <ul>
    <li>Product Listing Pages (PLP)</li>
    <li>Category Pages</li>
    <li>Recommendation Widgets</li>
  </ul>
  <p>
    This data powers 'Popular Products' and personalized 'Recommended For You'
    lists.
  </p>
</Accordion>

{" "}

<Accordion title="Add to Cart" description="Tracks cart additions">
  <p>
    Triggered when a shopper adds a product to the cart. For incremental
    additions, the event fires once per addition, with <code>qty</code>{" "}
    reflecting the added quantity.
  </p>
</Accordion>

  <Accordion title="Order" description="Tracks completed purchases">
    <p>Fires for each purchased product, capturing:</p>
    <ul>
      <li>Product ID</li>
      <li>Quantity</li>
      <li>Order details</li>
    </ul>
  </Accordion>
</AccordionGroup>
